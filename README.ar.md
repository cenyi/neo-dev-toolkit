## ✨ نظرة عامة

أداة JSON قوية تعمل في المتصفح، وتدعم أكثر من 20 نوعًا من التحويلات وتوليد الأكواد. معالجة 100٪ من جانب العميل - بياناتك لا تغادر متصفحك أبدًا！

### 🌟 الميزات الرئيسية

- 🎯 **الكشف التلقائي عن المدخلات**: يكتشف JSON ومعاملات URL و XML و YAML تلقائيًا
- 🔄 **20+ تحويل**: تحويل JSON ↔ YAML/XML/CSV/HTML/PDF والمزيد
- 💻 **توليد الأكواد**: TypeScript و Dart و Go و Rust و Python و C
- 🔍 **أدوات متقدمة**: مقارنة JSON واستخراج الحقول والتحقق من الصحة
- 🌐 **دعم ثنائي اللغة**: واجهة كاملة بالإنجليزية والصينية
- 📱 **محسّن للجوال**: يعمل بشكل مثالي على جميع الأجهزة
- 🔒 **الخصوصية أولاً**: 100٪ في المتصفح، لا رفع للخوادم
- ⚡ **سريع للغاية**: يعالج ملفات JSON تزيد عن 10 ميغابايت بسهولة

## 🚀 البداية السريعة

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/cenyi/neo-dev-toolkit.git

# الدخول إلى دليل المشروع
cd neo-dev-toolkit

# تثبيت التبعيات
npm install

# بدء خادم التطوير
npm run dev

# البناء للإنتاج
npm run build
```

## 🎯 الميزات

### عمليات JSON الأساسية

#### 1. **الكشف الذكي عن المدخلات** 🧠
يكتشف ويفك تلقائيًا:
- ✅ JSON القياسي
- ✅ معاملات URL
- ✅ مستندات XML
- ✅ ملفات YAML
- ✅ JSON مع التعليقات

#### 2. **تنسيق JSON والتحقق من الصحة** ✨
- تمييز بناء الجملة في الوقت الفعلي
- اكتشاف الأخطاء مع أرقام الأسطر
- تنسيق تلقائي عند اللصق

#### 3. **مقارنة JSON** 🔄
- مقارنة جنبًا إلى جنب
- تمييز بصري للفروقات:
  - 🟢 أخضر: مضاف
  - 🔴 أحمر: محذوف
  - 🟡 أصفر: معدّل

---

### 🔄 تحويلات تنسيق البيانات

- **JSON إلى YAML**: تنسيق YAML مع مسافات بادئة صحيحة
- **JSON إلى XML**: توليد XML ببنية وهروب صحيحين
- **JSON إلى CSV/Excel**: ملفات متوافقة مع Excel
- **JSON إلى HTML**: توليد HTML منسّق
- **JSON إلى جدول**: تصور بيانات JSON كجداول HTML جميلة
