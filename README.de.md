## ✨ Übersicht

Ein leistungsstarkes, browserbasiertes JSON-Toolkit mit 20+ Formatkonvertierungen, Codegenerierung und fortschrittlichen JSON-Operationen. 100% Client-seitige Verarbeitung - Ihre Daten verlassen nie Ihren Browser!

### 🌟 Highlights

- 🎯 **Smarte Eingabeverteilung**: Automatische Erkennung von JSON, URL-Parametern, XML, YAML & 15+ Formaten
- 🔄 **20+ Konvertierungen**: JSON ↔ YAML/XML/CSV/HTML/PDF/Tabelle & mehr
- 💻 **Codegenerierung**: TypeScript, Dart, Go, Rust, Python, C, JSON Schema
- 🔍 **Erweiterte Tools**: JSON-Vergleich, Diff, Feldextraktion, Validierung
- 🔒 **Privacy-First**: 100% browserbasiert, keine Server-Uploads

## 🚀 Schnellstart

### Installation

```bash
# Repository klonen
git clone https://github.com/cenyi/neo-dev-toolkit.git

# In Projektverzeichnis wechseln
cd neo-dev-toolkit

# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten
npm run dev

# Für Produktion bauen
npm run build
```

## 🎯 Features

### JSON-Basisoperationen

#### 1. **Smarte Eingabeverteilung** 🧠
Erkennt und parsed automatisch:
- ✅ Standard-JSON
- ✅ URL-Parameter (Query-Strings)
- ✅ XML-Dokumente
- ✅ YAML-Dateien
- ✅ JSON mit Kommentaren

#### 2. **JSON-Formatierung & Validierung** ✨
- Echtzeit-Syntaxhervorhebung
- Fehlererkennung mit Zeilennummern
- Automatisches Formatieren beim Einfügen

#### 3. **JSON-Vergleich & Diff** 🔄
- Seit-by-Seite-Vergleich
- Visuelle Diff-Hervorhebung:
  - 🟢 Grün: Hinzugefügt
  - 🔴 Rot: Entfernt
  - 🟡 Gelb: Geändert

---

### 🔄 Datenformatkonvertierungen

- **JSON zu YAML**: YAML-Format mit korrekter Einrückung
- **JSON zu XML**: XML mit korrekter Struktur
- **JSON zu CSV/Excel**: Excel-kompatible Ausgabe
- **JSON zu HTML**: Formatierter HTML mit Syntaxhervorhebung
- **JSON zu Tabelle**: JSON-Daten als schöne HTML-Tabellen
