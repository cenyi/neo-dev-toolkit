## ✨ Resumen

Un potente kit de herramientas JSON basado en navegador con 20+ conversiones de formato, generación de código y operaciones JSON avanzadas. ¡100% procesamiento del lado del cliente - sus datos nunca salen de su navegador!

### 🌟 Destacados

- 🎯 **Detección Inteligente de Entrada**: Detecta automáticamente JSON, parámetros URL, XML, YAML y 15+ formatos
- 🔄 **20+ Conversiones**: JSON ↔ YAML/XML/CSV/HTML/PDF/Tabla y más
- 💻 **Generación de Código**: TypeScript, Dart, Go, Rust, Python, C, JSON Schema
- 🔍 **Herramientas Avanzadas**: Comparación JSON, diff, extracción de campos, validación
- 🔒 **Privacidad Primero**: 100% basado en navegador, sin subidas al servidor

## 🚀 Inicio Rápido

### Instalación

```bash
# Clonar repositorio
git clone https://github.com/cenyi/neo-dev-toolkit.git

# Navegar al directorio del proyecto
cd neo-dev-toolkit

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build
```

## 🎯 Características

### Operaciones Básicas JSON

#### 1. **Detección Inteligente de Entrada** 🧠
Detecta y analiza automáticamente:
- ✅ JSON estándar
- ✅ Parámetros URL (cadenas de consulta)
- ✅ Documentos XML
- ✅ Archivos YAML
- ✅ JSON con comentarios

#### 2. **Formateo y Validación JSON** ✨
- Resaltado de sintaxis en tiempo real
- Detección de errores con números de línea
- Formateo automático al pegar

#### 3. **Comparación y Diferencias JSON** 🔄
- Comparación lado a lado
- Resaltado visual de diferencias:
  - 🟢 Verde: Agregado
  - 🔴 Rojo: Eliminado
  - 🟡 Amarillo: Modificado

### 🔄 Conversiones de Formato de Datos

- **JSON a YAML**: Formato YAML con sangría correcta
- **JSON a XML**: XML con estructura correcta
- **JSON a CSV/Excel**: Salida compatible con Excel

- **JSON a Tabla**: Visualiza datos JSON como tablas HTML hermosas
