## ✨ 概要

20以上のフォーマット変換、コード生成、高度なJSON操作をサポートする強力なブラウザベースのJSONツールキット。100%クライアントサイド処理 - データはブラウザから離れません！

### 🌟 ハイライト

- 🎯 **スマート入力検出**: JSON、URLパラメータ、XML、YAMLなどを自動検出
- 🔄 **20以上の変換**: JSON ↔ YAML/XML/CSV/HTML/PDF/テーブルなど
- 💻 **コード生成**: TypeScript、Dart、Go、Rust、Python、C、JSON Schema
- 🔍 **高度なツール**: JSON比較、diff、フィールド抽出、検証
- 🔒 **プライバシー優先**: 100%ブラウザベース、サーバーアップロードなし

## 🚀 クイックスタート

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/cenyi/neo-dev-toolkit.git

# プロジェクトディレクトリに移動
cd neo-dev-toolkit

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev

# 本番用にビルド
npm run build
```

## 🎯 機能

### JSON基本操作

#### 1. **スマート入力検出** 🧠
自動的に検出・解析:
- ✅ 標準JSON
- ✅ URLパラメータ
- ✅ XMLドキュメント
- ✅ YAMLファイル
- ✅ コメント付きJSON

#### 2. **JSONフォーマット & 検証** ✨
- リアルタイムシンタックスハイライト
- 行番号付きエラー検出
- ペースト時の自動フォーマット

---

### 🔄 データフォーマット変換

- **JSON→YAML**: 正しいインデントのYAML形式
- **JSON→XML**: 正しい構造のXML
- **JSON→CSV/Excel**: Excel互換出力
- **JSON→HTML**: シンタックスハイライト付きHTML
- **JSON→テーブル**: 美しいHTMLテーブルとしてJSONデータを可視化
