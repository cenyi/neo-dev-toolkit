## ✨ 개요

20개 이상의 포맷 변환, 코드 생성, 고급 JSON 작업을 지원하는 강력한 브라우저 기반 JSON 툴킷입니다. 100% 클라이언트 측 처리 - 데이터가 브라우저를 떠나지 않습니다!

### 🌟 하이라이트

- 🎯 **스마트 입력 감지**: JSON, URL 매개변수, XML, YAML 등 15개 이상의 포맷을 자동 감지
- 🔄 **20개 이상의 변환**: JSON ↔ YAML/XML/CSV/HTML/PDF/테이블 등
- 💻 **코드 생성**: TypeScript, Dart, Go, Rust, Python, C, JSON Schema
- 🔍 **고급 도구**: JSON 비교, diff, 필드 추출, 검증
- 🔒 **개인정보 우선**: 100% 브라우저 기반, 서버 업로드 없음

## 🚀 빠른 시작

### 설치

```bash
# 리포지토리 복제
git clone https://github.com/cenyi/neo-dev-toolkit.git

# 프로젝트 디렉토리로 이동
cd neo-dev-toolkit

# 종속성 설치
npm install

# 개발 서버 시작
npm run dev

# 프로덕션용 빌드
npm run build
```

## 🎯 기능

### JSON 기본 작업

#### 1. **스마트 입력 감지** 🧠
자동으로 감지 및 구문 분석:
- ✅ 표준 JSON
- ✅ URL 매개변수 (쿼리 문자열)
- ✅ XML 문서
- ✅ YAML 파일
- ✅ 주석이 있는 JSON

#### 2. **JSON 포맷팅 및 검증** ✨
- 실시간 구문 강조
- 줄 번호와 함께 오류 감지
- 붙여넣기时 자동 포맷팅

---

### 🔄 데이터 포맷 변환

- **JSON을 YAML로**: 올바른 들여쓰기가 있는 YAML 형식
- **JSON을 XML로**: 올바른 구조의 XML
- **JSON을 CSV/Excel로**: Excel 호환 출력
- **JSON을 HTML로**: 구문 강조가 있는 포맷된 HTML
- **JSON을 테이블로**: JSON 데이터를 아름다운 HTML 테이블로 시각화
