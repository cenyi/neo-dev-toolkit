## ✨ Visão Geral

Um poderoso kit de ferramentas JSON baseado em navegador com 20+ conversões de formato, geração de código e operações JSON avançadas. 100% processamento do lado do cliente - seus dados nunca saem do seu navegador!

### 🌟 Destaques

- 🎯 **Detecção Inteligente de Entrada**: Detecta automaticamente JSON, parâmetros URL, XML, YAML e 15+ formatos
- 🔄 **20+ Conversões**: JSON ↔ YAML/XML/CSV/HTML/PDF/Tabela e mais
- 💻 **Geração de Código**: TypeScript, Dart, Go, Rust, Python, C, JSON Schema
- 🔍 **Ferramentas Avançadas**: Comparação JSON, diff, extração de campos, validação
- 🔒 **Privacidade em Primeiro Lugar**: 100% baseado em navegador, sem uploads de servidor

## 🚀 Início Rápido

### Instalação

```bash
# Clonar repositório
git clone https://github.com/cenyi/neo-dev-toolkit.git

# Navegar para o diretório do projeto
cd neo-dev-toolkit

# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev

# Build para produção
npm run build
```

## 🎯 Recursos

### Operações Básicas JSON

#### 1. **Detecção Inteligente de Entrada** 🧠
Detecta e analisa automaticamente:
- ✅ JSON padrão
- ✅ Parâmetros URL (query strings)
- ✅ Documentos XML
- ✅ Arquivos YAML
- ✅ JSON com comentários

#### 2. **Formatação e Validação JSON** ✨
- Destaque de sintaxe em tempo real
- Detecção de erros com números de linha
- Formatação automática ao colar
