## ✨ Обзор

Мощный набор инструментов JSON на основе браузера с 20+ конвертациями формата, генерацией кода и расширенными JSON операциями. 100% обработка на стороне клиента - ваши данные никогда не покидают ваш браузер!

### 🌟 Основные моменты

- 🎯 **Умное Определение Ввода**: Автоматически определяет JSON, параметры URL, XML, YAML и 15+ форматов
- 🔄 **20+ Конвертаций**: JSON ↔ YAML/XML/CSV/HTML/PDF/Таблица и более
- 💻 **Генерация Кода**: TypeScript, Dart, Go, Rust, Python, C, JSON Schema
- 🔍 **Расширенные Инструменты**: Сравнение JSON, diff, извлечение полей, валидация
- 🔒 **Конфиденциальность Прежде Всего**: 100% на основе браузера, без загрузок на сервер

## 🚀 Быстрый Старт

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/cenyi/neo-dev-toolkit.git

# Перейти в каталог проекта
cd neo-dev-toolkit

# Установить зависимости
npm install

# Запустить сервер разработки
npm run dev

# Собрать для продакшена
npm run build
```

## 🎯 Функции

### Базовые Операции JSON

#### 1. **Умное Определение Ввода** 🧠
Автоматически обнаруживает и анализирует:
- ✅ Стандартный JSON
- ✅ Параметры URL (строки запроса)
- ✅ Документы XML
- ✅ Файлы YAML
- ✅ JSON с комментариями

#### 2. **Форматирование и Валидация JSON** ✨
- Подсветка синтаксиса в реальном времени
- Обнаружение ошибок с номерами строк
- Автоматическое форматирование при вставке

---

### 🔄 Конвертации Форматов Данных

- **JSON в YAML**: Формат YAML с правильными отступами
- **JSON в XML**: XML с правильной структурой
- **JSON в CSV/Excel**: Вывод совместимый с Excel
- **JSON в HTML**: Форматированный HTML с подсветкой синтаксиса
- **JSON в Таблицу**: Визуализация данных JSON как красивых HTML-таблиц
